## This repository is provided for code review purposes only. No usage rights are granted.

This is a 2D jaw generation in order to help user get a better view of their teeth. This can be replaced with the initial dental checkup and find/detecting
dental problem in early stages. this readme will be updated upon project process.

TechStack:
- Kotlin Multiplatform
- Camera : CameraX for Android and AVFoundation for ios
- ML : using a custom pytorch segmentation model converted to tflite
- Tflite: LiteRT on Android and tflite-objc on ios
